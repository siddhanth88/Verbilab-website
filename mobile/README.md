# Verbilab Mobile App

Flutter mobile application for Verbilab AI Content Creation Platform.

## Run locally

```bash
cd verbilab-web/mobile
flutter pub get
flutter run
```

## Structure

- `lib/main.dart` - Entry point and basic home screen
- `lib/screens/` - Screen widgets
- `lib/widgets/` - Reusable components

Matches the web version content and intent with a native mobile experience.
